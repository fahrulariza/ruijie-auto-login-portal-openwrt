# Ruijie Auto Login Captive Portal Script OpenWRT


FITUR SCRIPT INTELEGEN:
### ✅ Hanya login jika diperlukan:

    pengecekan internet

    Cek captive portal - hanya login jika ada portal Ruijie

    Tidak mengubah MAC address - karena terbukti merusak koneksi

### ✅ Multi-credentials:

    Coba login dengan Umum:PKMUMUM1 dulu

    Jika gagal, coba dengan Admin:PKMADMIN1

### ✅ Smart monitoring:

    Monitor koneksi internet setiap 5 menit

    Otomatis login jika internet mati dan ada captive portal

    Logging detail untuk debugging

### ✅ Aman:

    Tidak mengubah konfigurasi jaringan

    Restore state jika gagal

    Handle error dengan baik
